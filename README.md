# 🎨 ASCII ART PRO MAX

Convierte imágenes en arte ASCII directamente desde tu navegador.

## 🚀 Características

- 🖼️ Convertir imágenes a ASCII
- 📷 Cámara en vivo a ASCII
- 🌈 Modo color y blanco/negro
- 🎚️ Control de calidad, brillo y zoom

## 🌐 Demo

👉 https://TUUSUARIO.github.io/ascii-art-app/

## 🛠️ Tecnologías

- HTML
- CSS
- JavaScript
- Canvas API

## 📦 Cómo usar

1. Sube una imagen
2. Ajusta los controles
3. Descarga el resultado
